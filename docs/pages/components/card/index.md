### Simple card
```react-snippet
text-card
```
```jsx
<Card>
  <CardHeader>
    <CardTitle>Title goes here</CardTitle>
    <CardSubtitle>Subtitle</CardSubtitle>
  </CardHeader>
  <CardText>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
  </CardText>
  <CardActions>
    <Button compact >action 1</Button>
    <Button compact >action 2</Button>
  </CardActions>
</Card>
```
